B.L.I.N.K. ist ein NWT Projekt zur optischen Datenübertragung zwischen zwei Arduinos. Anstatt Funk oder Kabel zu verwenden, werden Informationen über Laserlicht übertragen. Ein Laser sendet kurze Lichtimpulse, ein Fototransistor empfängt sie, und der Arduino rekonstruiert daraus wieder lesbaren Text.

Die Kommunikation funktioniert über ein einfaches Protokoll: Beide Arduinos befinden sich im Ruhezustand mit dauerhaft eingeschaltetem Laser. Möchte einer senden, prüft er zuerst, ob der optische Kanal frei ist. Ist das der Fall, schaltet er den Laser für eine Sekunde aus, um ein Startsignal zu erzeugen. Der Empfänger erkennt diese Dunkelphase, synchronisiert sich und liest anschließend die Bits in festen Zeitabständen ein. Die Daten werden als 7‑Bit‑ASCII übertragen. Sobald das Licht länger konstant an bleibt, gilt die Nachricht als beendet.

Das System verhindert Kollisionen, indem ein Arduino nur sendet, wenn der optische Kanal frei ist. Dadurch kann immer nur einer senden, während der andere empfängt. Die gesamte Logik ist in klar getrennte Sender‑ und Empfängerklassen ausgelagert, während der Code im loop() minimal bleibt.

Für den Aufbau werden zwei Arduinos, zwei Laser oder LEDs, zwei Fototransistoren oder LDRs, passende Widerstände und jeweils zwei Taster benötigt. B.L.I.N.K. eignet sich gut für Experimente oder als Einstieg in andere Kommunikationsmethoden und zeigt, wie zuverlässig und flexibel Licht als Übertragungsmedium genutzt werden kann.
